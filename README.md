По всем заданиям можно использовать локальный хост или удаленный сервер.


Задание 1. Установка ПО:
    Задача: Напишите Ansible-плейбук, который устанавливает Nginx на удаленные серверы.
    Подсказка: Используйте модуль apt для систем Ubuntu/Debian или yum для систем CentOS/RHEL.

Задание 2. Копирование файла:
    Задача: Напишите Ansible-плейбук, который копирует локальный файл на удаленный сервер.
    Подсказка: Используйте модуль copy.

Задание 3. Настройка конфигурационного файла:
    Задача: Напишите Ansible-плейбук, который изменяет значение параметра в конфигурационном файле на удаленном сервере.
    Подсказка: Используйте модуль lineinfile для добавления или изменения строк в файлах.

Задание 4. Создание пользователя:
    Задача: Напишите Ansible-плейбук, который создает нового пользователя на удаленном сервере.
    Подсказка: Используйте модуль user.

Задание 5. Настройка файрвола:
    Задача: Напишите Ansible-плейбук, который открывает определенный порт в файрволе.
    Подсказка: Используйте модуль ufw для Ubuntu/Debian или firewalld для CentOS/RHEL.

Задание 6. Выполнение команды:
    Задача: Напишите Ansible-плейбук, который выполняет команду на удаленном сервере.
    Подсказка: Используйте модуль command или shell.

Задание 7. Создание директории:
    Задача: Напишите Ansible-плейбук, который создает новую директорию на удаленном сервере.
    Подсказка: Используйте модуль file.

Задание 8. Конфигурация SSH:
    Задача: Напишите Ansible-плейбук, который копирует ваш открытый SSH-ключ на удаленные серверы для авторизации без пароля.
    Подсказка: Используйте модуль authorized_key.

Задание 9. Установка Docker:
    Задача: Напишите Ansible-плейбук, который устанавливает Docker на удаленные серверы.
    Подсказка: Используйте модуль shell или command для выполнения команд установки Docker.

Задание 10. Управление сервисами:

Задача: Напишите Ansible-плейбук, который останавливает, запускает или перезапускает сервис на удаленных серверах.
Подсказка: Используйте модуль systemd или service.
